Android Development

This guide assumes you have already followed the the Common Install Steps

Initial Install Steps

N/A

Running App

  1. Open ./android folder in Android Studio

  2. Ensure React-Native server is running

     npm run develop

  3. Ensure build variant is set to debug

  4. Run build in Android Studio

Generating APK

  1. Open ./android folder in Android Studio

  2. Ensure build variant is set to either "qa" or "release" depending on desired build type

  3. Select "Build" -> "Build APK(s)" from Android Studio menu bar

    1. Builds can also be run with ./gradlew build, which generates all variants.

    2. Run ./gradlew jacocoTestReport to generate a test report.
