iOS

iOS Development

This guide assumes you have already followed the Starting Development guide

Initial Install Steps

  1. Install Cocoapods dependencies

    cd ios
pod install

Running App

  1. Open .xcworkspace file located in ./ios folder

  2. Ensure React-Native server is running

    npm run develop

  3. Ensure "Run" scheme in Xcode is set to "Debug"

  4. Build/Run in Xcode

Generating IPA / XCARCHIVE

Manually

  1. Generate updated React Native JS Bundle (run from project root)

    npm run bundle:ios

  2. Open .xcworkspace file located in ./ios folder

  3. Ensure "Archive" scheme is set to either "QA" or "Release" depending on desired build type

  4. Select "Product" -> "Archive" from Xcode menu bar

Automated (CI/CD)

This approach uses Fastlane and a few different Ruby gems

  1. Install bundler

    gem install bundler

  2. Install dependencies in the ./ios folder

    bundle install

  3. Generate the QA version of the app

    bundle exec fastlane qa_app  # qa_app, debug_app, release_app

  4. Run tests

    bundle exec fastlane test

Next Steps

Try adding a feature - Check out Adding Features or the README.md file located in your /app directory
