# Project Structure

```md
YOUR_APP/
    __tests__/        # React Native unit tests
    android/          # Android files
    app/              # React Native project code
    ios/              # iOS files
    www/              # Cordova root
    common.js         # Common imports for both iOS and Android
    index.ios.js      # React Native's iOS entry point
    index.android.js  # React Native's Android entry point
    app.json          # Project file
```
